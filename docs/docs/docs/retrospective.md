# Project Retrospective

## 1. Overview
This retrospective reflects on my experience building the “How to Make the Perfect Chocolate Chip Cookie” website. I talk about what went well, what was challenging, and what I learned while completing the project.

## 2. What Went Well
- Setting up the GitHub repository and docs folder was straightforward.
- Writing the content for the recipe and tips pages felt natural.
- Keeping the layout simple made the styling easier.
- Navigation between pages worked the first time after testing.

## 3. Challenges
- Making sure all files were in the correct folders took extra attention.
- Getting GitHub Pages to load correctly required checking file paths.
- Keeping the design consistent across multiple pages took some trial and error.
- Writing clear instructions for beginners made me rethink how I explain steps.

## 4. What I Learned
- How to organize a multi‑page website using HTML and CSS.
- How to use GitHub Pages to host a live site.
- How important clean navigation and simple styling are for readability.
- How to break a project into smaller steps so it feels manageable.

## 5. Final Thoughts
Overall, this project helped me understand how to plan, build, and publish a small website from start to finish. I feel more confident using GitHub, writing HTML/CSS, and organizing project files. The final site is simple, clear, and meets all the requirements.
