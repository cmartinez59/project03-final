# Project Plan

## 1. Overview
This plan explains how I will build my multi‑page website about making the perfect chocolate chip cookie. I will follow a simple workflow so each part of the project is organized and easy to complete.

## 2. Project Steps

### Step 1: Set Up the Repository
- Create the GitHub repository  
- Add a README.md  
- Create the docs/ folder  
- Add scope.md, plan.md, and retrospective.md  

### Step 2: Build the Website Structure
- Create index.html (homepage)  
- Create about.html (recipe or tips page)  
- Create style.css  
- Add navigation links between pages  

### Step 3: Add Content
- Write the homepage introduction  
- Add the recipe steps and ingredients  
- Add tips for better cookies  
- Insert images if needed  

### Step 4: Style the Website
- Set fonts, colors, and spacing  
- Make sure all pages use the same style.css  
- Keep the layout simple and readable  

### Step 5: Test the Website
- Check all links  
- Make sure images load  
- Test the site on GitHub Pages  
- Fix any layout issues  

### Step 6: Finalize the Project
- Review the checklist  
- Make sure all files are in the correct folders  
- Submit the GitHub repo link in Canvas  

## 3. Timeline
- **Day 1:** Create repo and documentation  
- **Day 2:** Build HTML pages  
- **Day 3:** Add content and images  
- **Day 4:** Style the site  
- **Day 5:** Test and finalize  

## 4. Tools Used
- GitHub  
- GitHub Pages  
- VS Code  
- HTML and CSS  

## 5. Expected Outcome
A clean, simple website that teaches beginners how to make great chocolate chip cookies, with working navigation, readable content, and consistent styling.
