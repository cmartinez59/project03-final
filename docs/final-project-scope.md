# Project Scope Statement

## Project Title
How to Make the Perfect Chocolate Chip Cookie

## 1. Project Purpose
The purpose of this project is to make a simple website that teaches people how to bake really good chocolate chip cookies. I want the site to be easy to follow so beginners can understand the ingredients, steps, and common mistakes.

## 2. Project Description
This will be a small multi‑page website built with HTML and CSS. The homepage will introduce the topic, the recipe page will show the ingredients and steps, and the tips page will explain things that help the cookies turn out better. The site will have simple navigation, clear sections, and a clean layout.

## 3. Desired Results
- A working multi‑page website  
- Easy‑to‑read instructions for beginners  
- Simple navigation between all pages  
- Helpful images to support the recipe  
- A final ZIP file with all required project files  

## 4. Exclusions
- No JavaScript  
- No backend or database  
- No video tutorials  
- No advanced animations or frameworks  

## 5. Constraints
- Must use only HTML and CSS  
- Must follow the course file structure  
- Must be hosted on GitHub Pages  
- Must be finished within the project timeline  

## 6. Risks & Mitigation
- **Risk:** Images might not load correctly on GitHub Pages  
  **Fix:** Use correct file paths and test the live site  
- **Risk:** Layout might look different on smaller screens  
  **Fix:** Keep the design simple and flexible  

## 7. Success Criteria
- All pages load correctly on GitHub Pages  
- Navigation works on every page  
- Content is clear and beginner‑friendly  
- Website meets all checklist requirements  
- All documentation is in the docs/ folder
